Git tutorial
- master
- create branch develop from master
- create branch feature/update_readme from develop

Flow:
1 - Checkout new branch from develop
2 - Coding
3 - Commit code (local repository)
4 - Push code (remote repository: github, gitlab, ....)
5 - Create pull request/merge request
6 - Review code
7 - Fix code (if needed)
8 - Merge code
9 - Checkout to develop
10 - Pull code
- return to step 1
